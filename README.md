# Prestation des enseignants d'une université

Application hybride pour la gestion de préstation d'enseignement dans une université

## Pré-requis
- React Native en frontend
- Laravel API en backend
- Axios pour relier le back et le front
- Expo pour l'utilisation mobile

## Fonctionnalités
- Gestion des prestations
- Tableau de bord des prestations
- Statistiques et représentation graphique
- Interface mobile responsive
